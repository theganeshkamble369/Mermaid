Mermaid.js: Visualizing Diagrams with Text

Learn how to create diagrams using text-based syntax with Mermaid.js

Introduction

Mermaid.js is a JavaScript library that allows you to create diagrams and visualizations using text-based syntax. In this blog post, we’ll explore how Mermaid.js can help you generate flowcharts, sequence diagrams, Gantt charts, and more with just a few lines of text.

Why Use Mermaid.js?

Mermaid.js simplifies the creation of diagrams and allows for easy integration with various platforms like Markdown, Jupyter Notebooks, and HTML. It's ideal for generating complex diagrams without needing a graphical tool.

Installation & Setup

To get started with Mermaid.js, follow these steps:

  1. Install Mermaid.js via npm or use the CDN in your HTML file.
  2. For npm: npm install mermaid
  3. For CDN: 
    <script type="module">
                        import mermaid from 'https://cdn.jsdelivr.net/npm/mermaid@10/dist/mermaid.esm.min.mjs';
                        mermaid.initialize({startOnLoad:true});
                    </script>

Code Examples with Outputs

Example 1: Flowchart

This is an example of a simple flowchart using Mermaid.js:

```mermaid
graph LR
    A[Start] --> B[Process 1]
    B --> C[Process 2]
    C --> D[End]
```

Example 2: Sequence Diagram

Here’s a sequence diagram demonstrating interactions between Alice and John:

```mermaid
sequenceDiagram
    Alice->>+John: Hello John, how are you?
    John-->>-Alice: I'm good, thanks for asking!
```

Example 3: Gantt Chart

This is an example of a Gantt chart that can be used for project management:

```mermaid
gantt
    title A Gantt Diagram
    dateFormat  YYYY-MM-DD
    section Section
    A task           :a1, 2025-02-01, 30d
    Another task     :after a1  , 20d
```

Example 4: Pie Chart

Mermaid.js can also create pie charts. Here's an example:

```mermaid
pie
    title Distribution of Market Share
    "Product A" : 35
    "Product B" : 25
    "Product C" : 40
```

Example 5: Class Diagram

Here’s a class diagram showing the relationship between classes in an object-oriented system:

```mermaid
classDiagram
    ClassA <|-- ClassB
    ClassA : +int x
    ClassB : +int y
    ClassB : +int sum()
```

Use Cases

Mermaid.js can be used in a variety of scenarios:

  • Documentation: Easily embed diagrams in Markdown files.
  • Project Planning: Create Gantt charts to visualize timelines.
  • Software Development: Visualize interactions with sequence diagrams.
  • Education: Explain complex workflows and processes in a simple way.

Conclusion

Mermaid.js is a powerful tool for generating diagrams in a text-based format. It’s simple to use and integrates well with many platforms like Markdown, Jupyter Notebooks, and more. Whether you're documenting code, planning a project, or visualizing workflows, Mermaid.js can help you create clear and concise diagrams.
